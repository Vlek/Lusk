# Lusk
[Unity-based](https://unity.com/) Orc-themed top-down RPG based on 2007ish freeshard era [Ultima Online](https://en.wikipedia.org/wiki/Ultima_Online).

This game seeks to replicate the feeling of being an evil RPer on the freeshard [UOHybrid](http://uohybrid.com/) during the peek time period as a single-player game.

The main goal at this point is to add all features necessary to have fort defense and Orcish raids on human settlements.

## Contributing

If you can create UO-style pixel art for the game, please reach out to me by creating an issue with a way to contact you and a sample of your work.

For now, I wish to do this without code contributions.

However, if you have something you would like me to look in to, please create a new issue for it with as much detail as you can give.
